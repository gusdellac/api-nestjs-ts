## Docker

### Construir y ejecutar en modo desarrollo

```bash
docker-compose -f docker-compose.dev.yml up --build -d
```


### Ejecutar en modo desarrollo con imagen preconstruida

```bash
docker-compose -f docker-compose.dev.yml up -d
```

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
